什么是注解

Java 中的注解类型 (Annotation Types，下称注解)可以理解成是对代码的一种特殊标记，我们可以把注解标记在类、方法、变量、参数等 Java 元素上。这个标记可以给编写代码的人看，但是它主要还是提供给程序本身进行解析。

也就是说，注解本身其实不能起到作用，还需要其他程序去主动解析注解来采取相应的动作。如果没有相应的解析代码，那么注解就起不到任何作用。
